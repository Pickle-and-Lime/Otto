updateInvitation

(
  • householdId
  • inviteeEmail
  • accept
)
Promise

Defined in server/controllers/userController.js:151

Accepts or rejects the invitation.

Accepting will change the invitee's HouseholdId to the creator's householdId, and add the invitee's email to the creator household's users list.

Both accept and reject will remove the household's invite from the User's invites array, and from the household's sentInvites array

Parameters:

  • householdId String
  • inviteeEmail String
  • accept Boolean

Returns:

Promise:

Callback is supplied with invitee's User object
